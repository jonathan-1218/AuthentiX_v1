# Authentic Frontend Dashboard

A React + Vite dashboard for monitoring sensor inputs, reviewing operational chat updates, and viewing infographic-style charts.

## Features

- Real-time styled sensor monitoring layout
- Summary KPI cards
- Trend and alert visualizations with Recharts
- Operations chat panel UI
- Responsive dashboard design

## Scripts

- `npm run dev` - start the local development server
- `npm run build` - build for production
- `npm run preview` - preview the production build locally

## Main files

- `src/App.jsx` - dashboard layout and mock sensor/chat data
- `src/index.css` - dashboard styling
- `src/main.jsx` - React entry point

## Notes

The current version uses mock data so the UI is ready for live sensor API integration.
